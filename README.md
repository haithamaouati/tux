## Installation

1. Update & upgrade packages
    ```
    pkg update -yy && pkg upgrade -yy
    ```
    
2. Install git package
    ```
    pkg install git -yy
    ```

3. Clone the repository
    ```
    git clone https://github.com/haithamaouati/tux.git
    ```
4. Change to tux directory
    ```
    cd tux
    ```
    
5. Change the access mode
    ```
    chmod +x tux.sh
    ```

## Usage

6. Run the script

    ```
    bash tux.sh
    ```
